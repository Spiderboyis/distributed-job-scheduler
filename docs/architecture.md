## Data Flow: Job Lifecycle

```
                 User/API
                    │
                    ▼
            ┌──────────────┐
            │   Create Job  │
            │   (REST API)  │
            └──────┬───────┘
                   │
     ┌─────────────┼─────────────┐
     │             │             │
     ▼             ▼             ▼
 immediate      delayed      recurring
     │             │             │
     ▼             │             ▼
  QUEUED           │        Scheduler loop
     │             │        checks cron →
     │             ▼        creates job →
     │         SCHEDULED         │
     │             │             │
     │    (wait for scheduled_at)│
     │             │             │
     │             ▼             │
     └──────► QUEUED ◄───────────┘
                   │
                   ▼
    ┌────────────────────────────┐
    │  Worker: SELECT FOR UPDATE │
    │  SKIP LOCKED               │
    │  (atomic claim)            │
    └────────────┬───────────────┘
                 │
                 ▼
             CLAIMED
                 │
                 ▼
             RUNNING
                 │
          ┌──────┴──────┐
          │             │
          ▼             ▼
      COMPLETED      FAILED
                        │
                  ┌─────┴─────┐
                  │           │
              retry?      exhausted
                  │           │
                  ▼           ▼
            back to       DEAD (DLQ)
             QUEUED
```
